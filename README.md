## Overview

**gaga**, a brandnew java TCP server + client software for *CSE 589 project*.

### Usage
#### 


	java Echoer <tcp-port> <udp-port>

Here are some commands:

- info
- connect <tcp-port> <udp-port>
- show
- send <conn-id> <message>
- sendto <ip-address> <udp-port> <message>
- disconnect <conn-id>
